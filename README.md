# Allele Specificity Analysis for 'Perspectives on Codebook: sequence specificity of uncharacterized human transcription factors' paper

## Contents
* `AS_CHS_GHTS` — scripts and notebooks for allele specificity analysis of ChIP-seq and GHT-SELEX data.
* `common` — scripts and data used in both approaches (phenotype annotation, motif annotation, etc).
* `motifs` — Codebook motifs (PWMs, P-Value thresholds, Logos).
* `peaks` — scripts for downloading and processing protein binding peaks and TOPs.

## EXTRAs
